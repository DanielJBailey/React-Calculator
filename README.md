# React Calculator

A simple calculator created using React. The goal for this calculator is to allow users to use their mouse or numeric keypad from their keyboard to operate this calculator.

### Required operations
* Add
* Subtract 
* Multiply
* Divide 
* Percentages
* Square Roots
* Backspace
* Clear/reset
